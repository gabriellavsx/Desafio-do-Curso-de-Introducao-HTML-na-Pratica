Estrutura Básica do HTML

Voltar

Em um documento HTML é necessário utiliar uma estutura composta pelas tags html, head, e body. Dentro da tag HTML ficará todas as outras tags usadas. Dentro da tag head ficará o cabeçalho do documento com informações do documento para o navegador e outros serviços web, é obrigatório o uso da tag title dentro de head. A tag body é responsável por todo o corpo de um documento HTML, tudo que é exibido para a janela, sendo assim, tododo o conteúdo visível de um site.

Falando sobre Tags

Voltar

Em um documento HTML, existem várias tags, como por exemplo a tag i que é responsável por deixar o texto em italico. A tag strong que deixa o texto em negrito. A tag input que é usada para criar controles interativos para formulários baseados na web para receber dados do usuário. A tag img que faz parte do conjunto de elementos da linguagem de marcação utilizados para exibir conteúdos visuais em uma página.

Atributos das Tags

Voltar

E um documento HTML, existem vários atributos, temos como exemplo o id que é um identificados de elementos. O atributo style é usada no para colocar elementos do CSS direto nos elementos do HTML. O atributo class é usada para padronizar os elementos. O atributo type se refere ao tipo de do input. O atributo scr é usado na tag img para referenciar o caminho para a imagem que será usao. O atributo width é usado na tag img para definir a largura da imagem.

Textos

Voltar

E um documento HTML, existem várias tags para o textos, as peincipais são: p que representa um paragrafo dentro da tag body. A tag h representa o titulo da página e vária de h1 até h6, o h1 será o título maior e o h6, o menor. A tag blockquote é usada para referenciar uma sitação. A tag sup deixa op textpo na parte de cima da linha. A tag u é usado para deixar o txto sublinhado.

Listas Ordenadas e Não Ordenadas

Voltar

E um documento HTML, existem as Listas Ordenadas e Não Ordenadas. As Listas Ordenadas são representadas pela tela ol. As Listas Ordenadas são representadas pela tela ul. A tag para criar uma liosta normal é li.

Links

Voltar

E um documento HTML, existe a tag a que é usada para ralizar uma âncora ou hyperlink. Dentro dessa tag é possível usar o atributo href que faz referência do link que a pessoa ao clicar será referenciada.
